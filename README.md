# Estudos em Java ![Java](https://img.shields.io/badge/java-%23ED8B00.svg?style=for-the-badge&logo=openjdk&logoColor=white)

Repositório dedicado aos meus estudos da linguagem **Java**, contendo exercícios, práticas e pequenos testes realizados durante o aprendizado.

## 🔹Objetivo

Praticar conceitos fundamentais de programação e reforçar conhecimentos adquiridos durante o curso.

## 🔹Conteúdos estudados

* Lógica de programação
* Array List
* Funções
* Programação Orientada a Objetos (POO)
* Classes e Objetos
* Encapsulamento
* Herança e Polimorfismo
* Interfaces
* Abstração

## 🔹Estrutura do repositório

Os exercícios estão organizados em pastas conforme os temas ou atividades propostas durante os estudos.

## 🔹Observação

Este repositório faz parte do meu processo de aprendizado e evolução na programação, não são projetos completos e detalhados.
